# Пикассо — Конструктор открыток

Веб-приложение для быстрого создания корпоративных поздравительных открыток Сбера. Сотрудник выбирает фон, добавляет текст и скачивает готовую открытку в PNG или PDF за несколько кликов.

## Быстрый старт

Откройте `index.html` в браузере — сборка не требуется.

## Структура проекта

```
Picasso-NEW/
├── index.html              # Единственная HTML-страница
├── css/
│   └── style.css           # Все стили
├── js/
│   └── app.js              # Вся логика
└── assets/
    └── images/
        ├── preset_image_1-7.png   # Готовые фоны
        └── sber-logo.png          # Логотип Сбера
```

## Как работает приложение

Три последовательных шага:

### Шаг 1 — Фон
Пользователь выбирает один из 7 готовых фонов или загружает своё фото. При загрузке своего фото оно отображается в той же ячейке сетки с тем же градиентным оформлением.

### Шаг 2 — Текст
Заполняются поля открытки:
- **Заголовок** — например «С днём рождения!»
- **Обращение** — имя получателя
- **Текст поздравления** — основной текст, поле авторастягивается под объём, есть кнопка «Очистить»
- **Подпись** — имя отправителя

Дополнительные инструменты:
- **Готовые тексты** — 10 заготовленных поздравлений в аккордеоне
- **Помощь ИИ** — генерирует промт для ChatGPT по заданным параметрам (пол, должность, стиль)
- **Логотип Сбербанка** — переключатель Да/Нет, по умолчанию выключен

Правая панель (десктоп) показывает предпросмотр открытки в реальном времени.

### Шаг 3 — Скачать
- **PNG** — высокое качество, 3× разрешение от экранного
- **PDF** — автоматически определяет ориентацию

После скачивания доступны кнопки «Вернуться и изменить текст» и «Создать новую открытку».

## Дизайн открытки

Открытка состоит из двух независимых блоков:

```
┌─────────────────────┐
│                     │  ← Блок изображения (aspect-ratio 4:3)
│  [лого Сбера]       │     Фото с градиентным затемнением снизу
│                     │
├─────────────────────┤  ← Плавный переход (градиент)
│ С днём рождения!    │
│ Иван Иванович!      │  ← Блок текста (изумрудный градиент)
│ Текст поздравления… │     Высота динамическая
│ Подпись             │
└─────────────────────┘
```

Блоки визуально бесшовны: нижняя часть изображения растворяется в том же изумрудно-зелёном цвете, с которого начинается текстовый блок.

## Добавление новых фонов

1. Положить изображение в `assets/images/`
2. Добавить путь в массив `TEMPLATE_IMAGES` в начале `js/app.js`

```js
const TEMPLATE_IMAGES = [
  'assets/images/preset_image_1.png',
  // ... добавить новый путь сюда
];
```

## Добавление готовых текстов

Добавить строку в массив `TEXT_PRESETS` в начале `js/app.js`:

```js
const TEXT_PRESETS = [
  'Текст первого шаблона…',
  // ... добавить новый текст сюда
];
```
